Person—blood group Code (Data Element Concept)

Definition

Any of various classes into which human blood can be divided according to immunological compatibility,

based on the presence or absence of specific antigens on red blood cells.

 

If you department is refering this data element,
you can register in MDR by selecting the checkbox
